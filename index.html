What is LocNotes?

LocNotes is a location-based note-taking app that helps you create and recall notes based on where you are and when you need them.

Key Features

  • üìç Location-Based: Notes automatically appear when you're within 10 meters of where they were created
  • ‚è∞ Time-Scheduled: Set notes to appear at specific dates and times (5-minute resolution)
  • üè∑Ô∏è Hashtags: Tag notes with #hashtags for easy organization
  • üîó Note References: Link notes together using #123 syntax
  • üéØ Smart Sorting: Notes are ordered by location-time proximity

How to Use

  1. Create a Note: Tap the ‚ûï button, write your note, and save
  2. Location Nicknames: Give places friendly names like "Home" or "Office"
  3. Tag Locations: Attach hashtags to locations to auto-show tagged notes when you're there
  4. Search: Use the search box to find notes by content or hashtags
  5. Export/Import: Backup your data as JSON from Settings

  • ‚úì Your notes never leave your device

Important Caveats

‚ö†Ô∏è Private/Incognito Mode Warning

If you use LocNotes in a private or incognito browser window, all your notes will be permanently deleted when you close the window.

Data Storage: Your notes are stored in your browser's IndexedDB. They persist across normal browsing sessions but will be lost if you:

  • Use private/incognito mode
  • Clear browser data
  • Uninstall the browser
  • Use a different browser or device

Recommendation: Regularly export your data (Settings ‚Üí Export Data) to create backups!

Location Permissions

LocNotes requires location access to function. Your location data:

  • Is only stored locally in your browser
  • Is never sent to any server
  • Is used only to match notes to your current position

Tips

  • Use hashtags like #work #meeting to organize notes
  • Attach hashtags to locations for automatic filtering
  • Reference other notes with #123 (note ID numbers)
  • Export your data regularly as a backup
  • Notes are sorted by relevance - most important appear first
